# 真实二维码样本说明

## 来源

1. **维基百科样本** (wiki_qr_*.png)
   - 来自维基百科公开图片
   - 清晰度高，标准样式

2. **API生成样本** (generated_*.png)
   - 使用QR Server API生成
   - 不同颜色主题
   - 不同尺寸
   - 包含各种真实数据类型

3. **错误纠正级别测试** (ecc_*.png)
   - L: 7% 数据可恢复
   - M: 15% 数据可恢复
   - Q: 25% 数据可恢复
   - H: 30% 数据可恢复

4. **不同数据量测试** (data_*.png)
   - 短数据
   - 中等长度数据
   - 长数据
   - 长URL

## 使用建议

这些图片适合用于:
- 功能测试
- 算法验证
- 性能基准测试
